# 📞 Salla Call Center Performance Analytics
### لوحة معلومات Power BI المتقدمة لتحقيق التميز التشغيلي

---

## 🌟 نظرة عامة على المشروع
تم تصميم هذا المشروع خصيصاً لتحليل وتحسين أداء مركز اتصال شركة **Salla**. قمت بتحويل سجلات ضخمة تتجاوز **2 مليون مكالمة** إلى لوحة تحكم تفاعلية تساعد في اتخاذ قرارات تشغيلية دقيقة.

---

## 🏗️ هيكل لوحة التحكم (Dashboard Structure)
تم تقسيم المشروع إلى 3 واجهات رئيسية لضمان شمولية التحليل:

1. **ملخص تنفيذي (Executive Summary):** لمراقبة المؤشرات الكلية مثل إجمالي المكالمات ومستوى الخدمة (Service Level) بنسبة 90%.
2. **أداء الوكلاء (Agent Performance):** تحليل تفصيلي لإنتاجية الموظفين، معدل رضا العملاء (CSAT)، وأوقات الاستجابة.
3. **النظرة التشغيلية (Project Overview):** تتبع الاتجاهات الزمنية (Trends) والتنبؤ بحجم المكالمات المستقبلي لتحسين توزيع الموارد.

---

## 🛠️ المهارات والتقنيات المستخدمة
* **تحليل البيانات:** استخدام **Power Query** لتنظيف وتجهيز ضخمة البيانات.
* **نمذجة البيانات:** بناء علاقات فعالة بين الجداول لضمان سرعة التقارير.
* **معادلات DAX:** كتابة مقاييس متقدمة لحساب نسبة رضا العملاء ومعدلات التخلي (Abandon Rates).
* **التصميم البصري (UI/UX):** اختيار Dark Theme عصري لتسهيل قراءة البيانات وتقليل إجهاد العين.

---
